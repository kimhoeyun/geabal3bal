# 👣 개발세발
개발 천리길도 세 걸음부터! 당신의 개발을 도와줄 서비스, 개발세발.

---

## 1. 서비스 요약
매일 다른 **기술 정보 뉴스**, **알고리즘 테스트**, **CS 이론 리뷰** 세 가지 분야의 정보를 제공하고,  
매주 새로운 **오픈소스 코드 리뷰**를 통해 개발자의 성장을 돕는 **메일 구독 서비스**.


## 2. 서비스 방향

### ✅ 사용자에게 제공할 가치 
- **지식 제공 + 실전 경험**


## 3. 주요 기능

1. **기술 정보 뉴스레터**  
   - 최신 기술 동향 및 AI 이슈  

2. **알고리즘 이론**  
   - 관련 문제 링크 제공  

3. **CS 이론**  
   - 핵심 CS 개념 정리  

4. **오픈소스 코드 리뷰**  
   - 실무 코드 분석 및 개선 포인트 제공  



## 4. 제공 방식
- **메일 구독 서비스** 형태로 콘텐츠 발송
